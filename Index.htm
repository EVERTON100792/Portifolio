Minha Jornada de Aprendizado

Uma trajetória de dedicação contínua para transformar código em soluções incríveis.

A Base Sólida: HTML5 e CSS3

Iniciei minha jornada focado em construir a estrutura fundamental da web, dominando a criação de sites semanticamente corretos, essenciais para SEO.

Dando Vida às Páginas: JavaScript

Fui além do visual estático, mergulhando no JavaScript para criar interações que engajam o usuário, como animações e menus dinâmicos.

Design que se Adapta: Layouts Responsivos

Com os fundamentos sólidos, foquei em dominar Flexbox e CSS Grid para garantir que os sites sejam perfeitos em qualquer dispositivo (Mobile-First).

Evolução Contínua: Foco em React.js e Performance

Atualmente, estou a aprofundar os meus estudos em React.js para criar interfaces ainda mais dinâmicas e otimizadas, sempre com foco em performance.
