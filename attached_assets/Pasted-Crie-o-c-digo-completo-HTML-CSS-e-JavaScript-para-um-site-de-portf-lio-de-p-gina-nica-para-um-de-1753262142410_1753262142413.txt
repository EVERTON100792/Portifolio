Crie o código completo (HTML, CSS e JavaScript) para um site de portfólio de página única para um desenvolvedor web júnior chamado Everton.

**Requisitos Técnicos:**

1.  **Tecnologias:** Use apenas HTML, CSS e JavaScript puros, sem frameworks.
2.  **Arquivos Separados:** Crie três arquivos: `index.html`, `style.css` e `script.js`.
3.  **Responsividade:** O layout deve ser totalmente responsivo, aplicando o conceito de "Mobile First".
4.  **Performance:** Utilize a API "Intersection Observer" em JavaScript para que as animações iniciem apenas quando os elementos entram na tela.

**Estrutura e Conteúdo da Página:**

1.  **Seção de Abertura (Herói):**
    * Título: "Everton | Desenvolvedor Web Criativo".
    * Subtítulo: "Transformando ideias em experiências digitais interativas."
    * Animação de Fundo: Use a biblioteca `particles.js` para criar um fundo com partículas que reagem ao movimento do mouse.

2.  **Seção "Sobre Mim":**
    * Uma foto profissional do Everton e um texto de apresentação.
    * Animação: Efeitos de `fade-in` (surgimento suave) para a foto e o texto ao rolar a página.

3.  **Seção "Linha do Tempo de Estudos":**
    * Crie uma linha do tempo vertical e interativa.
    * Marcos: "Fev/2025: Fundamentos de HTML e CSS", "Abr/2025: Dominando JavaScript e o DOM", "Jun/2025: Controle de versão com Git e GitHub".
    * Futuro: "Próximos Passos: Explorar React.js e Node.js".
    * Animação: Conforme o usuário rola, a linha se "desenha" e os marcos aparecem.

4.  **Seção "Meus Projetos":**
    * Crie uma galeria com 3 cards de projetos fictícios.
    * **Cada card deve conter:**
        * Uma imagem de pré-visualização do projeto.
        * Título do projeto.
        * Breve descrição e tecnologias usadas (ícones de HTML, CSS, JS).
        * **Dois botões claros e funcionais:**
            * Um botão principal **"Ver Site ao Vivo"**. Use um link de placeholder como `https://#`.
            * Um botão secundário **"Ver o Código"** que leva para o GitHub. Use um link como `https://#`.
    * Animação: Efeito de `hover` nos cards, aumentando-os sutilmente.

5.  **Elemento Fixo:**
    * Um **botão flutuante do WhatsApp** no canto inferior direito, com uma animação de "pulso" contínua para chamar atenção.
    * O link do botão deve ser: `https://api.whatsapp.com/send?phone=SEUNUMERO&text=Olá,%20Everton!%20Vi%20seu%20portfólio%20e%20gostaria%20de%20fazer%20um%20orçamento.`

**Instruções Finais:**
* Comente o código HTML, CSS e JavaScript para explicar as seções principais e as lógicas de animação.
* O design geral deve ser limpo, moderno e com um tema de cores coeso (ex: tons de azul escuro, branco e um verde para destaque no WhatsApp).
